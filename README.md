# 🎙️ Stress Detection from Speech using Wav2Vec2

## 📌 Overview

An AI-powered system that detects whether a person is **Stressed** or **Unstressed** from speech recordings using **Facebook's Wav2Vec2 Transformer model**.

---

## 🚀 Features

✅ Speech-based Stress Detection
✅ Transformer-Based Deep Learning (Wav2Vec2)
✅ Automatic Audio Processing
✅ Real-Time Prediction Support
✅ Frontend Integration Ready

---

## ⚙️ Workflow

🎤 Audio Input
⬇️
🔄 Preprocessing (16kHz WAV)
⬇️
🧠 Wav2Vec2 Feature Extraction
⬇️
🤖 Classification Model
⬇️
📊 Stress Prediction
⬇️
✅ Stressed / ❌ Unstressed

---

## 📂 Dataset

* 📁 Total Samples: **280**
* 🎙️ Audio Length: **≤ 5 sec**
* 🏷️ Classes:

  * Stressed
  * Unstressed

* 📚 Epochs: 3
* 📦 Batch Size: 4
* 📉 Learning Rate: 1e-5
* ⚡ Optimizer: AdamW
* 💻 Platform: Google Colab (Tesla T4)
